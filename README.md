# Mortality Rate Trend Analysis

## 📌 Objective
To analyze mortality rate trends across regions, age groups, gender, and years to identify patterns and key insights.

---

## 🛠 Tools Used
- Python (Pandas)
- Power BI

---

## 📊 Dataset Information
The dataset contains 200 records with 15 columns including:
- Year, Region, Country, State
- Gender, Age Group, Cause of Death
- Deaths, Population
- Hospital Type, Urban/Rural
- Income Level, Healthcare Access
- Environmental factors (Temperature, Pollution)
